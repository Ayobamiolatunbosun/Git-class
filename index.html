introduction to git

Git is an open source version control susytem that is used to track the changes made to a code

advantages of using git

  • fast and efficient
  • distributed version control
  • enhanced collaboration
  • compactibility
  • data integrity

Configuringb git

git config --global user.name "Ayobami Olatunbosun"

git config --global user.email "olatunbosuncars@gmail.com"

Basic git command

  • git init: used to initialise a git repository
  • git add . :used to stage changes made to our code
  • git status: used to check the status of our code
  • git commit -m"commit message" :this is used to save the staged changes with a descriptive caption or massage
  • git remote add origin: this is used to connect our local to out remote repo on github
  • git push origin main: this is used to push your code from local repo to remote repo on github
